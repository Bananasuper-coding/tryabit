## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📱 Features

### Core Learning System
- **28 Complete Lessons** across 4 weeks
- **Micro-learning** with 5-6 phrases per day
- **Spaced repetition** vocabulary cards
- **Shadowing practice** with speech synthesis
- **Real grammar** in native style
- **Quick writing** with AI-style feedback
- **Daily missions** and challenges

### Gamification
- **XP & Level System** - Earn XP for every action
- **Daily Streaks** - Keep your streak alive!
- **10 Mini-Games** - Learn while playing
- **Achievements** - 30+ unlockable achievements
- **Weekly Boss Battles** - Review challenges
- **Progress Tracking** - Visual heatmap

### Design
- **Dark UI** with glassmorphism effects
- **Smooth animations** using Framer Motion
- **Mobile-first** responsive design
- **Bottom navigation** for easy thumb access
- **PWA Support** - Install on any device

## 🌐 Deployment

### GitHub Pages

```bash

```

### Vercel / Netlify

Simply connect your GitHub repository and deploy!

## 📖 Usage

1. **Start Learning**: Click "Today's Lesson" on dashboard
2. **Complete Sections**: Work through phrases, vocabulary, dialogue, grammar
3. **Play Games**: Reinforce learning with mini-games
4. **Earn XP**: Complete lessons and games to gain XP
5. **Maintain Streak**: Learn every day to keep your streak!
